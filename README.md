A comprehensive, web-based Project Management System designed to support efficient project planning, tracking, and risk management in enterprise environments.

🏦 Recognition: This system is being developed as an independent initiative for internal use within Dashen Bank SC, where it has received positive recognition for its practical value in improving workflow efficiency and project coordination.

🚀 Overview
The Project Management System provides a centralized platform for managing the full project lifecycle, including planning, execution, monitoring, and reporting. Built with a modular and scalable architecture, it supports multiple users, structured workflows, and real-time interaction across different project components.

🎯 Core Objective
Improve internal project management efficiency within enterprise environments

Provide a centralized and structured system for project coordination

Apply software engineering best practices in a real-world organizational context

🛠️ Technologies Used
Frontend
HTML5 – Semantic markup structure

CSS3 – Responsive and modern styling

JavaScript – Client-side interactivity

jQuery – Simplified DOM manipulation and AJAX calls

Backend
PHP – Server-side business logic implementation

Database
MySQL – Relational database for structured data management

Architecture
The system follows a standard multi-tier web architecture:

Presentation Layer: Interactive UI built with JavaScript/jQuery

Application Layer: Business logic implemented in PHP

Data Layer: MySQL database for persistent storage

✨ Key Features
📋 Project Planning & Tracking
Create and manage project timelines

Track milestones and overall progress

Monitor project phases and deliverables

👥 User Management Module
Register new users with role assignments

Activate/deactivate user accounts

Delete users from the system

Comprehensive role-based access control

📁 Project Profile Management
Create and maintain project profiles

Assign project managers and team members

Track project-specific metadata and documentation

Monitor project status and health indicators

🐞 Issue Tracking System
Log and categorize project issues

Assign issues to team members

Track resolution status and priority levels

Maintain issue history and comments

🧪 Test Case Management
Create and organize test cases

Link tests to project requirements

Track testing progress and results

Manage test execution cycles

📅 Event & Task Management
Schedule project events and meetings

Assign tasks to individual users

Set deadlines and priority levels

Monitor task completion status

💰 Budget Management
Track project budgets and allocations

Monitor expenditures vs. planned budget

Generate financial reports

Manage cost categories and approvals

⚠️ Risk Management Module
Identify and document project risks

Assess risk probability and impact

Define mitigation strategies

Monitor risk status and triggers

👥 Role-Based Access Control (RBAC)
Granular permission management

User role definitions (Admin, Manager, Team Member, Viewer)

Module-level access restrictions

Secure authentication and session management

📊 Dashboard & Reporting System
Individual Module Dashboards:

User Management Dashboard

Project Profile Dashboard

Task Management Dashboard

Issue Tracking Dashboard

Risk Management Dashboard

Unified Dashboard:

Consolidated project insights

Real-time metrics and KPIs

Cross-module data visualization

Executive summary views

🔔 Notification System
System-wide notifications for key events

Email notifications (optional integration)

Real-time alerts for task assignments

Status change notifications

⚙️ Installation & Setup
Prerequisites
Web server (Apache/Nginx recommended)

PHP 7.4 or higher

MySQL 5.7 or higher

Web browser with JavaScript enabled

Step-by-Step Installation
Clone the repository

bash
git clone https://github.com/MikiDevGuy/project-management-system.git
Navigate to project directory

bash
cd project-management-system
Configure your local server

For XAMPP: Move project to htdocs folder

For WAMP: Move project to www folder

For LAMP: Move project to /var/www/html/

Database Setup

Open phpMyAdmin (or MySQL CLI)

Create a new database (project_management)

Import the provided SQL file:

bash
mysql -u username -p project_management_db < database/project_management.sql
Or use phpMyAdmin import feature

Configure Database Connection

Navigate to config/database.php

Update database credentials:

php
define('DB_HOST', 'localhost');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
define('DB_NAME', 'project_management');
Run the Application

Open your browser

Navigate to: http://localhost/project-management-system

Default login credentials:

Username: admin

Password: admin123 (change immediately after first login)
